## Usage

Simply open `index.html` in a modern web browser to run the calculator. No additional setup or dependencies required.

### Basic Operations

- Numbers (0-9)
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Modulus (%)
- Clear (CLR)
- Delete (DEL)
- Calculate (=)

- Implements secure calculation evaluation by sanitizing input
